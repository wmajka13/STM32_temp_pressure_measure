Miniprojekt - Odczytanie pomiaru temperatury z czujnika DS18B20, wyświetlenie jej na wyświetlaczu siedmiosegmentoweym oraz przesłanie przez UART.

Co można jeszcze zrobić:
- Dodać odczyt ciśnienia,
- Dodać możliwość zmiany tego co się wyświetla (temperatura lub ciśnienie) przyciskiem,

Użyto STM32 Nucleo-F411RE oraz KA-Nucleo-Multisensor.
